Detailed Description

Provides a text item that can be added to a DrawingScene.

To set the item's text, call the setCaption() function. The caption() function returns the current text.

Rendering options for the item can be controlled through properties of the item's style(). The text item supports all of the font, text brush, and text alignment style properties.

DrawingTextItem provides a reasonable implementation of boundingRect(), shape(), and isValid(). The render() function draws the text using the item's associated pen and brush.

DrawingTextItem::DrawingTextItem ( )

Create a new DrawingTextItem with default settings.

This function creates one DrawingItemPoint object to serve as an anchor point for the text.

This function fills in the item's style() with default values for the following properties. The default values are pulled from the style's DrawingItemStyle::defaultValues() if present, otherwise DrawingRectItem attempts to use reasonable initial values for each property:

  • DrawingItemStyle::TextColor
  • DrawingItemStyle::TextOpacity
  • DrawingItemStyle::FontName
  • DrawingItemStyle::FontSize
  • DrawingItemStyle::FontBold
  • DrawingItemStyle::FontItalic
  • DrawingItemStyle::FontUnderline
  • DrawingItemStyle::FontOverline
  • DrawingItemStyle::FontStrikeThrough
  • DrawingItemStyle::TextHorizontalAlignment
  • DrawingItemStyle::TextVerticalAlignment
